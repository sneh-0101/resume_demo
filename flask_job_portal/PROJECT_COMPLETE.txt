## ğŸ¤– AI ENGINE CAPABILITIES

### Resume Parser
- Input: PDF file path
- Process: pdfplumber page extraction
- Output: Full text content

### Skill Extractor
- Database: 100+ technical + soft skills
- Method: Regex pattern matching
- Output: Set of extracted skills

### TF-IDF Calculator
- Input: Resume text + Job description
- Method: scikit-learn vectorization
- Output: Cosine similarity (0-1)

### Skill Matcher
- Input: Resume skills + Job requirements
- Method: Set intersection
- Output: Match percentage

### Hybrid Scorer
- TF-IDF Score: 40% weight
- Skill Score: 60% weight
- Output: Overall score (0-100)
- Match levels: Excellent/Good/Fair/Poor

## ğŸ“Š DATABASE SCHEMA

### Users Table (8 columns)
```sql
id, username (UNIQUE), email (UNIQUE),
password_hash, role, first_name, last_name,
phone, bio, company, is_active, created_at, updated_at
```

### Jobs Table (11 columns)
```sql
id, title, description, requirements, location,
salary_min, salary_max, job_type, company,
recruiter_id (FK), is_active, created_at, updated_at
```

### Resumes Table (8 columns)
```sql
id, user_id (FK), filename, filepath,
extracted_text, extracted_skills (JSON),
is_primary, created_at, updated_at
```

### Applications Table (11 columns)
```sql
id, job_id (FK), user_id (FK), resume_id (FK),
status, match_score, matched_skills (JSON),
missing_skills (JSON), cover_letter,
applied_at, updated_at
```
